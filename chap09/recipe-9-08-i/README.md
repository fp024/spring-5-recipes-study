## 진행

##### 레시피 9-08-i

* 스프링 관리 트랜젝션 적용

  * `@Transactional` 을 적용해서 DAO의 메서드가 단순해졌다.
  * DAO에 `@Repository`는 이미 붙이고 있었는데..
    * 일관된 예외 처리를 하려면 스프링 `DataAccessException`으로 변환해야하는데, 이렇게 하려면 `@Repository`를 DAO클래스에 붙여야한다.
